# Aditya's Portfolio

A premium personal portfolio website for an AI & Machine Learning Engineer.

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** TailwindCSS
- **Animations:** Framer Motion
- **Smooth Scroll:** Lenis
- **Language:** TypeScript

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Customization

- Update your information in the component files
- Replace profile images in `/images` directory
- Modify projects in `components/ProjectsSection.tsx`
- Adjust colors in `tailwind.config.ts`

## Build for Production

```bash
npm run build
npm start
```
